CSSのみで構成するグラデーション

ライナーグラディエントについて

linear-gradient

linear-gradientはbackgroundプロパティで使用する値の一つで、cssで背景を2色以上使用してクラデーション効果を生み出す事が出来ます。
開始位置から終了位置までそれぞれに色を指定し、上から下、左から右など自由に方向を決める事で様々な表現ができます。

具体的には 要素にbackgroundのプロパティを設定する際に開始位置(0%)の色を決めて、次に終了位置の色を決める(100%)。続いて方向(to top,to bottom,90deg など)を決めると、2色のグラデーションを作る事が出来ます。

（例）グラデーションの基本的な記入方法 

グラデーションを掛けたい要素のセレクタ {
  background: linear-gradient(to top, #cd9cf2 0%, #f6f3ff 100%); /* 0%: #cd9cf2、100%: #f6f3ff、下から上に向かって変化していくグラデーション */
}

基本的なグラデーションの角度や色数を変えたもをそれぞれ記載しています。 

.box.gradecolor1 {
  background: linear-gradient(180deg, #2af598 0%, #009efd 100%);
}
.box.gradecolor2 {
  background: linear-gradient(to top, #cd9cf2 0%, #f6f3ff 100%);
}
.box.gradecolor3 {
  background: linear-gradient(to right, #f78ca0 0%, #f9748f 20%, #fd868c 60%, #fe9a8b 100%); /* 0% から 20%、20% から 60%、60% から 100% と左から右に向かって変化していくグラデーション */
}
.box.gradecolor4 {
  background: linear-gradient(to bottom, #6fb8d9 0% 30%, #f4c14e 40% 70%, #d36950 80% 100%);
}

グラデーションの角度指定の注意点

linear-gradientを角度なしで色のみで設定した場合(例：linear-gradient(#f00, #00f))、#f00が上に#00fが下にきます。

linear-gradientを角度指定を0°にした場合(例：linear-gradient(0deg, #f00, #00f))、角度を指定していない状態から、角度0°を指定するので、色の配置は反転し上下逆になります。

グラデーションの使い方

linear-gradientを使用することで、複雑な配色が可能になります。これにより、デザインからコーディングに作業が移行する際、CSSによる再現性が高まり、画像を切り出す作業が軽減されます。

See the Pen グラデーションパターン01 by Daizo Kaneko (@D-kaneko) on CodePen.

(linear-gradient(#111 0%, #111 50%, #cb0000 50%, #cb0000 100%))は、開始点（0%）から（50%）までの色（#111）を決め、次に（50%）から（100%）までの色（#cb0000）を決めることで、範囲を明確に区別し、双方の色がグラデーションすることなく境界線を作成することができます。

WEBサイト全体の背景色を2色に分けたい時や、今まで画像の切り出しが必要だった2色以上使う装飾などはcssで可能になります。

See the Pen Untitled by Daizo Kaneko (@D-kaneko) on CodePen.

linear-gradientは、画像のURLを指定するだけで画像の上にグラデーションを描画する事が出来ます。

通常時linear-gradientとhover時のlinear-gradientを用意すれば、ホバー時に切り替えることが出来たり、キーフレームを使う事で経過時間に応じてグラデーションの色を変える事も可能です

See the Pen Untitled by Daizo Kaneko (@D-kaneko) on CodePen.

フリーグラデーションを作りたい場合、使用する色数分のdivを用意しそれを一つのdivで囲みます。各要素(div)に対しサイズや位置、色を設定し、filter: blur(設定値);を使用し各要素をぼかすことで、フリーグラデーションを作る事が出来ます。

グラデーションの繰り返し処理について
