CSSのみで構成するグラデーション

ライナーグラディエントについて

linear-gradient

linear-gradientはbackgroundプロパティで使用する値の一つで、cssで背景を2色以上使用してクラデーション効果を生み出す事が出来ます。
開始位置から終了位置までそれぞれに色を指定し、上から下、左から右など自由に方向を決める事で様々な表現ができます。

具体的には 要素にbackgroundのプロパティを設定する際に開始位置(0%)の色を決めて、次に終了位置の色を決める(100%)。続いて方向(to top,to bottom,90deg など)を決めると、2色のグラデーションを作る事が出来ます。

（例）グラデーションの基本的な記入方法 

グラデーションを掛けたい要素のセレクタ {
  background: linear-gradient(to top, #cd9cf2 0%, #f6f3ff 100%); /* 0%：#cd9cf2、100%：#f6f3ff、下から上に向かっていくグラデーション */
}

基本的なグラデーションの方向や色数を変えたもをそれぞれ記載しています。 

.box.gradecolor1 {
  background: linear-gradient(180deg, #2af598 0%, #009efd 100%);
}
.box.gradecolor2 {
  background: linear-gradient(to top, #cd9cf2 0%, #f6f3ff 100%);
}
.box.gradecolor3 {
  background: linear-gradient(to right, #f78ca0 0%, #f9748f 20%, #fd868c 60%, #fe9a8b 100%);
}
