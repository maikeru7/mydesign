CSSのみで構成するアニメーション

トランジションのプロパティについて

CSS Transition

トランジションとは、要素の1つの状態から別の状態への変化をスムーズにアニメーションさせるために使用されるCSSプロパティです。
一般的に、マウスを要素に重ねる（ホバーする）などのユーザーのアクションに応じて、要素の特定の属性（例：色、サイズ、位置など）が変化する場合に利用されます。
具体的な使用例として、ボタンの色がマウスがホバーされたときに変化するように設定することができます。

具体的には セレクターに対し、トランジションというプロパティに動きの詳細を決める値を組みこむ、従来通りの構成で要素を動かす事ができます。

（例） トランジション有りと無しの比較
トランジションがない場合、青いボックスにホバーをすると瞬時に指定された幅に変化します。
それに比べて、トランジションがある場合は、対象物に対しtransitionで指定した通りに変化します。

トランジション無し

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
}
.blue-box:hover {
  width: 200px;
}

トランジション有り

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: all 1s ease-in-out; /* transitionのプロパティに対し、時間と動きの値を入れるだけ */
}
.blue-box:hover {
  width: 200px;
}

transitionプロパティについて

transitionは、transition-property、transition-duration、transition-timing-function、transition-delayの4つのプロパティに分けて設定できます。
それぞれのプロパティを個別に設定することで、より細かい動きを表現することができ、以下のように記入する事ができます。

4つのプロパティを分割して使用する場合の記法

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition-property: all; /* アニメーションを適用するプロパティ（ここでは、変化する対象は全てとなります） */
  transition-duration: 1s; /* アニメーションの時間（秒） */
  transition-timing-function: ease-in-out; /* アニメーションのイージング関数 */
  transition-delay: 1s; /* アニメーションの遅延（秒） */
}
.blue-box:hover {
  width: 200px;
}

クラス名blue-boxにホバーした場合、1秒後に幅が200pxに変化します。この変化に1秒かけて、全てのプロパティがease-in-outというイージング関数でアニメーションされます。

transitionのみ使用して、ひとまとめにした記法

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: all 1s ease-in-out 1s; /* ※transition-property、transition-duration、のプロパティの順番を間違えないように注意 */
}
.blue-box:hover {
  width: 200px;
}

4つのtransitionプロパティをひとまとめに書くと下記のようになります。
transition: all 1s ease-in-out 1s;
transition-property(all)、transition-duration(1s)、transition-timing-function(ease-in-out)、transition-delay(1s) の順番で記述しています。

プロパティ名読み方説明
transition-propertyトランジション・プロパティ変化が適用される CSS のプロパティを指定します。
transition-durationトランジション・デュレーション変化が始まって終わるまでの時間を指定します。
transition-timing-functionトランジション・タイミング・ファンクション変化の度合いを指定します。
transition-delayトランジション・ディレイ変化が始まる時間を指定します。
transitionトランジション上記の 4つのプロパティを一括で指定できる、ショートハンド・プロパティです。

アニメーションのプロパティについて

CSS Animation

アニメーション: 複数の状態を指定して、時間経過とともに変化するアニメーションを作成できます。
CSS Transitionより細かい設定が可能なので、jQueryなどのJavaScriptを使うなく、CSSのみで実装することが可能になります。

具体的には アニメーションを適用したい要素に対して、@keyframesを使って時間に応じた動作を定義し、複雑な動きを実現します。

（例）基本的な記入方法 

動かしたい要素のセレクタ {
  animation: アニメーションネーム 持続時間 イージング関数 遅延時間 繰り返し回数 方向;
}

@keyframes アニメーションネーム {
  0% {
    width: 120px;
  }
  100% {
    width: 360px;
  }
}

動かしたい要素のセレクタの中にanimationプロパティを指定します。
その中でアニメーションネームとそれ以外のプロパティを設定します。
次に、@keyframesルール内で、アニメーションネームとアニメーションの進行に関する細かい動作を定義します。

動かしたい要素に対して、アニメーションネーム：sampleAnm、持続時間：2.5s、イージング関数：linear、遅延時間：0s、を設定したものを実際に動かしてみると…（ページを再読み込みすると再現されます） 

.box {
  margin-bottom: 10px;
  background-color: #125585;
  color: #fff;
  animation: sampleAnm 2.5s linear 0s; //アニメーションネーム 持続時間 イージング関数 遅延時間
  width: 360px;
}

@keyframes sampleAnm { //@keyframesにて、アニメーションネーム指定します。
  0% {
    width: 120px;
  }
  100% {
    width: 360px;
  }
}

その他のプロパティについて

CSS Transform

トランスフォームは、要素を移動、変形や回転させることができるCSSプロパティです。単体で使用したり、CSS TransitionやCSS Animationと組み合わせたりすることで、複雑な動きや滑らかなアニメーションを実現できます。

具体的には CSS TransitionやCSS Animationと組み合わせて使用するためには、以下のような記述になります。

（例1）transformプロパティとの組み合わせ 

動かしたい要素のセレクタ {
  transition: transform（またはall） 0.3s ease; /* transformプロパティの変化をアニメーション化 */
}

動かしたい要素のセレクタ:hover {
  transform: rotate(45deg) scale(1.2); /* マウスホバー時のアニメーション設定 */
}

要素をマウスカーソルで覆った際に変化させたい場合、擬似クラス(:hover) に対し transformプロパティ内で rotate や scale を設定し、動かしたい要素のセレクタに transition プロパティを設定します。
transition プロパティは、要素の状態が変化する際のアニメーションを定義します。アニメーションの対象（ここではtransform もしくはallを使用）、時間、タイミングなどを指定することで、滑らかな動きを実現できます。

（例2）animationプロパティとの組み合わせ 

動かしたい要素のセレクタ:hover {
  animation: fadeawayAnime 2s ease; /* マウスホバー時のアニメーション設定 */
}

@keyframes fadeawayAnime{
  0% {
    transform: scale(1, 1);
  }
  25% {
    transform: scale(1.08, 1.08); /* アニメーションの進み具合によって、transformを使い要素の形状を変化させる */
  }
  50% {
    transform: scale(1, 1); 
  }
  100% {
    opacity: 0; 
  }
}

hoverで動く場合、animationプロパティは擬似セレクタにanimationプロパティにアニメーションネームやアニメーションに掛かる時間、イージング関数などを書きます。 

.tran-anim .box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: transform（またはall） 0.3s ease; /* transition内のvalueにtransformで設定したものとアニメーションにかかる時間とイージング関数 */
}

.tran-anim .box:hover {
  transform: rotate(45deg) scale(1.2); /* transform内のvalueは45°回転と1.2倍に拡大 */
}

transformプロパティを追加する事で動きに幅が広がります。

.keyframes-anime .box {
  width: 100px;
  height: 100px;
  background-color: blue;
}

.keyframes-anime .box:hover {
  animation: fadeawayAnime 2s ease;
}

@keyframes fadeawayAnime{
  0% {
    transform: scale(1, 1); /* @keyframes0%の時、大きさはそのまま */
  }
  25% {
    transform: scale(1.08, 1.08); /* @keyframes25%の時、少しだけ大きくなります */
  }
  50% {
    transform: scale(1, 1); /* @keyframes50%の時、下の大きさに戻ります */
  }
  100% {
    opacity: 0; /* そしてそのまま、スッと消えるアニメーションを、2秒かけてeaseで再現 */
  }
}

@keyframesやイージング関数、経過時間を使用する事で動きを細かく設定する事が出来ます。
