CSSトランジションとCSSアニメーションについて

トランジションのプロパティについて

CSS Transition

トランジションとは、要素の1つの状態から別の状態への変化をスムーズにアニメーションさせるために使用されるCSSプロパティです。
一般的に、マウスを要素に重ねる（ホバーする）などのユーザーのアクションに応じて、要素の特定の属性（例：色、サイズ、位置など）が変化する場合に利用されます。
たとえば、ボタンの色がマウスがホバーされたときに変化するように設定することができます。

具体的には セレクターに対し、トランジションというプロパティに動きの詳細を決める値を組みこむ、従来通りの構成で要素を動かす事ができます。

（例） トランジション有りと無しの比較
トランジションがない場合、青いボックスにホバーをすると瞬時に指定された幅に変化します。
それに比べて、トランジションがある場合は、対象物（ここでは横幅）、4

トランジション無し

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
}
.blue-box:hover {
  width: 200px;
}

トランジション有り

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: all 1s ease-in-out; /* transitionのプロパティに対し、時間と動きの値を入れるだけ */
}
.blue-box:hover {
  width: 200px;
}
プロパティ名読み方説明
transition-durationトランジション・デュレーション変化が始まって終わるまでの時間を指定します。
transition-propertyトランジション・プロパティ変化が適用される CSS のプロパティを指定します。
transition-timing-functionトランジション・タイミング・ファンクション変化の度合いを指定します。
transition-delayトランジション・ディレイ変化が始まる時間を指定します。
transitionトランジション上記の 4 つのプロパティを一括で指定できる、ショートハンド・プロパティです。

transitionプロパティについて

transitionは、transition-duration、transition-property、transition-timing-function、transition-delayの4つのプロパティに分けて指定できます。それぞれ のプロパティを個別に設定 することで、より細かい動きを表現することができ、以下のように記入する事ができます。

4つのプロパティ

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition-duration: 1s; /* アニメーションの時間（秒） */
  transition-property: all; /* アニメーションを適用するプロパティ（ここでは全てのプロパティ） */
  transition-timing-function: ease-in-out; /* アニメーションのタイミング関数 */
  transition-delay: 1s; /* アニメーションの遅延（秒） */
}
.blue-box:hover {
  width: 200px;
}

クラス名blue-boxにホバーした場合、1秒後に幅が200pxに変化します。この変化に1秒かけて、全てのプロパティがease-in-outというタイミング関数でアニメーションされます。

transitionのみ

.blue-box {
  width: 100px;
  height: 100px;
  background-color: blue;
  transition: all 1s ease-in-out 1s; /* ※transitionのプロパティの順番を間違えないように注意 */
}
.blue-box:hover {
  width: 200px;
}

4つのtransitionプロパティをひとまとめに書くとこうなります。
transition: all 1s ease-in-out 1s; transition-property(all)、transition-duration(1s)、transition-timing-function(ease-in-out)、transition-delay(1s) の順番で記述します。

アニメーションのプロパティについて

CSS Animation

アニメーション: 複数の状態を指定して、時間経過とともに変化するアニメーションを作成できます。たとえば、要素が回転したり、移動したりするなど。
